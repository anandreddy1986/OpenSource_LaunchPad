# Contributors

Welcome to the **OpenSource LaunchPad** contributor wall! 

If you attended one of our college tech talks — or simply want to make your first ever open source contribution — add your name below and submit a Pull Request. That's it. You're now an open source contributor.

---

## How to Add Your Name

1. **Fork** this repository (click the "Fork" button at the top right)
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/OpenSource_LaunchPad.git
   ```
3. **Create a branch:**
   ```bash
   git checkout -b add-my-name
   ```
4. **Add your name** to the table below (follow the format)
5. **Commit and push:**
   ```bash
   git add contributors.md
   git commit -m "Add [Your Name] to contributors"
   git push origin add-my-name
   ```
6. **Open a Pull Request** from your fork to this repo

Need help? Check the [step-by-step guide](https://opensource.guide/how-to-contribute/) or raise an [issue](https://github.com/anandreddy1986/OpenSource_LaunchPad/issues).

## Why This Matters

This might look like a small table edit — but you just:

- Forked a repository
- Created a branch
- Made a change
- Wrote a commit message
- Opened a Pull Request
- Got your work reviewed and merged

That's the **exact same workflow** used by engineers at every major tech company in the world. Your GitHub profile now shows a real open source contribution. Well done.

## What's Next?

Now that you've made your first contribution, keep going:

- Browse [goodfirstissue.dev](https://goodfirstissue.dev/) for beginner-friendly issues
- Explore [up-for-grabs.net](https://up-for-grabs.net/) for projects looking for help
- Try the [demo-bugs repo](https://github.com/anandreddy1986/demo-bugs) for a guided bug-fixing exercise
- Apply to [Google Summer of Code](https://summerofcode.withgoogle.com/) or [Outreachy](https://www.outreachy.org/)

**Aim for 1-2 contributions per month. In 6 months, your GitHub profile will speak louder than any resume.**
